# Put+Call 组合策略研究报告

**策略：** UTurnBottomShortPut（底部 Mix UTurn → 卖 Bull Put Spread）
**+** UTurnBottomShortCall（顶部 Mix UTurn → 卖 Bear Call Spread）

**回测区间：** 2024-01-01 ～ 2026-03-28（约 27 个月）
**研究品种：** ES、NQ、QQQ、SPY、IWM、SMH、GS、JPM、TSM
**合约乘数：** ES=50、NQ=20、股票期权=100（已计入收益率计算）
**最大回撤定义：** (equity − peak_equity) / peak_equity，与优化器一致

### 资金假设说明

- **单策略：** 初始资金 $100,000（保守基准，实际保证金远低于此）
- **组合（Iron Condor）：** 资本基数 = max(平均 put 保证金, 平均 call 保证金)。
  Iron Condor 最大亏损只发生在一侧，券商仅要求较宽一腿的保证金。
  保证金 = (spread_width − net_credit) × multiplier，已扣除收到的权利金。

---

## 1. 研究方法

对每个品种，测试 2×2 = 4 种区间搭配：

| 组合 | Call 周期 | Put 周期 |
|------|-----------|---------|
| C_M5+P_M5 | M5 | M5 |
| C_M5+P_M15 | M5 | M15 |
| C_M15+P_M5 | M15 | M5 |
| C_M15+P_M15 | M15 | M15 |

**选择原则：**
- 月度 PnL 相关系数 < 0 → 有对冲价值，优先选负相关中 Sharpe 最高者
- 全部正相关 → 无对冲价值，直接选同周期（M5+M5 或 M15+M15）中表现最优者

**评估指标：** 月度相关系数、年化收益率（%）、Sharpe 比率、最大回撤率（%）、亏损月数

## 7. 关键结论

1. **Iron Condor 资本效率极高：** 实际保证金远低于 $100,000（ES 约 $23k，股票期权 $1k～$7k），
   年化收益率以真实保证金口径计算后大幅提升，充分体现了卖方价差策略的资本优势。

2. **负相关规律：** ES、IWM、GS、JPM 的 Put 与 Call 策略在某些周期组合下呈负相关，
   具备对冲价值；NQ、QQQ、SPY、TSM 全部正相关，组合仅起叠加收益作用，无对冲效果。

3. **最优周期搭配：** 在有负相关的品种中，**C_M15+P_M5** 反复出现为最优组合（ES、GS、JPM、IWM），
   即 Call 用 M15 周期、Put 用 M5 周期。

4. **Put 策略更稳定：** ShortPut 亏损月普遍少于 ShortCall，
   Put_M5 在 GS、JPM 均实现 0 亏损月，是组合的压舱石。

5. **SMH 特殊性：** 在 M5/M15 限制下 Call 表现弱（Sharpe < 2.5）。
   若放开周期限制，Call_H1+Put_M5 可达 Sharpe=4.96、0 亏损月（另见全周期分析）。

6. **NQ 正相关较强：** 四种组合相关系数均为正（0.010～0.184），
   Put+Call 同向波动，组合风险分散效果有限，需单独监控回撤。
